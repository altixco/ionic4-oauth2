# Mercado Campesino

> App for selling farmer association's products

## Build Setup ##

``` bash
# Install Ionic
npm install -g ionic

# Install dependecies
npm install

# Serve the application
ionic serve

# Run app on Android device
ionic cordova run android

# Build dev app for Android
ionic cordova build android

# Run unit tests
npm run test

# Run e2e tests
npm run e2e

```
